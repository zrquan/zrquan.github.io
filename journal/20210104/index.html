00:30 git 强制推送 subtree

git checkout master # 切换到主分支
git subtree split --prefix dist -b gh-pages # 将 subtree 分割出一条本地分支, 包含本地的修改记录
git push -f origin gh-pages:gh-pages # 强制推送分割出来的分支
git branch -D gh-pages # 删除本地分支
