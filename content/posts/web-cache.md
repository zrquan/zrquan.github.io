## Web 缓存简介 {#web-缓存简介}

Web 缓存技术的应用其实非常普遍，只是它对于用户来说比较透明，容易被忽略。缓存的目的是加快资源文件的访问速度，让用户浏览页面更加流畅，同时减轻 Web 服务器的负载。无论是 CDN、负载均衡、反向代理，甚至一些开发框架，都会应用缓存技术来缓存部分资源的响应。
